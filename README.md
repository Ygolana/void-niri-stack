# Void Niri Stack

A modular Wayland stack installer for Void Linux featuring:

- Niri compositor
- Quickshell shell UI
- Noctalia shell integration
- GPU auto-detection
- Clean modular installer framework

Built for clean reproducible setups on Void Linux.

---

## 🚀 One-Line Install (SSH)

```bash
git clone git@github.com:Ygolana/void-niri-stack.git
cd void-niri-stack
chmod +x install.sh
./install.sh --all
